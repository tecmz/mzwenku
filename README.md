# 魔众文库系统

魔众文库系统基于文档系统知识，建立平台与领域，打造流量、用户、付费和变现的闭环，帮助您更好的搭建文库系统。

## 运行环境


```
操作系统
Linux/Unix 或&nbsp;&nbsp;Windows
软件环境
Laravel 5.1的运行环境
Apache/Nginx , PHP 5.5.9+ / PHP 7.0 , MySQL 5.0+
```
